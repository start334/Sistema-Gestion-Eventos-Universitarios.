# Funcionalidades implementadas

El Sistema de Gestión de Eventos Universitarios cuenta con funcionalidades destinadas a estudiantes y organizadores.

## Registro de usuarios

Permite crear nuevas cuentas mediante el ingreso de:

- Nombre completo.
- Correo institucional.
- Contraseña.
- Rol del usuario.

El sistema también valida los datos ingresados antes de completar el registro.

## Inicio de sesión

Permite que los usuarios registrados ingresen al sistema utilizando su correo institucional y contraseña.

## Selección de imagen de perfil

El usuario puede utilizar la imagen recomendada según su rol o cargar una fotografía propia.

## Gestión de eventos

El organizador puede:

- Crear eventos.
- Editar eventos.
- Eliminar eventos.
- Consultar los eventos registrados.

Para crear un evento se solicitan datos como nombre, fecha, hora, lugar, cupos y descripción.

## Consulta de eventos

Los usuarios pueden visualizar los eventos disponibles y buscar actividades por nombre o lugar.

## Inscripción a eventos

Los estudiantes pueden inscribirse en eventos disponibles cuando existan cupos.

La inscripción queda registrada dentro del sistema y puede consultarse posteriormente.

## Almacenamiento local

El sistema utiliza LocalStorage para conservar usuarios, eventos, inscripciones e imágenes de perfil dentro del navegador.
